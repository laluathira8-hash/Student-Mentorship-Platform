Student Mentorship Platform
----
Description
This is a full stack web application which is built using React.js for the frotend and Node.js with Express for the backend along with MongoDB as the database. The application enables users to perform CRUD operations and demostrates version control, CI/CD integration and cloud and devOps. 
----

- NODE.JS
- Express.js
- MongoDB
- EC2
- CI/CD
